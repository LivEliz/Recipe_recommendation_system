# Recipe Recommendation System

A simple and intuitive Python-based recipe recommendation system that suggests recipes based on provided ingredients.

Description
This project helps users find suitable recipes based on the ingredients they have. It uses basic data processing and matching logic to recommend appropriate recipes. Ideal as a mini project to understand how recommendation logic works using Python.

Built With
- Python
- pandas
- CSV data file

Features
- Accepts a list of ingredients as input
- Matches the input with recipes in the dataset
- Outputs recommended recipes based on available ingredients
